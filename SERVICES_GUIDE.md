# Руководство по работе с услугами MAXCAPITAL Bot

## Описание функционала

Бот теперь умеет:
1. Распознавать интерес клиента к услугам компании
2. Консультировать клиентов по интересующим услугам
3. Собирать данные клиента (ФИО и номер телефона)
4. Создавать лиды в Bitrix24 после сбора данных
5. Вести статистику в PostgreSQL

## Услуги компании

1. **VENTURE CAPITAL (Венчурный капитал)** - Инвестирование в стартапы (seed round и A round)
2. **HNWI Consultations** - Консультации для частных лиц с крупным капиталом (от $1 млн)
3. **REAL ESTATE (Недвижимость)** - Поиск и приобретение недооцененных объектов недвижимости
4. **CRYPTO (Криптовалюта)** - OTC сделки, консультирование по портфелям и майнингу
5. **M&A (Слияния и поглощения)** - Консультирование и сопровождение сделок
6. **PRIVATE EQUITY** - Выкуп бизнес-единиц для экспортных рынков
7. **Relocation Support** - Поддержка при релокации, получение статуса за рубежом
8. **ЗАРУБЕЖНЫЕ БАНКОВСКИЕ КАРТЫ** - Оформление карт через банки-партнеры

## Процесс работы бота

1. **Приветствие** - Бот приветствует клиента и показывает список услуг
2. **Распознавание интереса** - Бот определяет, какие услуги интересуют клиента
3. **Консультация** - Бот консультирует клиента по интересующим услугам
4. **Подтверждение** - Когда клиент готов, бот просит предоставить ФИО и телефон
5. **Сбор данных** - Бот извлекает ФИО и телефон из сообщений клиента
6. **Создание лида** - После сбора всех данных бот создает лид в Bitrix24

## Настройка базы данных

1. Создайте базу данных PostgreSQL:
```sql
CREATE DATABASE maxcapital_bot;
```

2. Выполните SQL скрипт для создания таблиц:
```bash
psql -U postgres -d maxcapital_bot -f database_schema.sql
```

3. Настройте переменные окружения в `.env`:
```
DB_HOST=localhost
DB_PORT=5432
DB_NAME=maxcapital_bot
DB_USER=postgres
DB_PASSWORD=your_password
```

## Статистика

Статистика ведется в таблицах:
- `bot_users` - пользователи бота
- `bot_sessions` - сессии (обращения)
- `bot_messages` - сообщения в сессиях
- `service_interests` - интерес к услугам

Метрики:
- **total_visitors** - сколько людей обратились к боту
- **total_interested** - сколько людей заинтересовались услугами
- **total_leads** - сколько людей стали лидами в Bitrix24

## API для статистики

GET `/statistics` - получить общую статистику
GET `/statistics/user/<user_id>` - получить статистику по пользователю
